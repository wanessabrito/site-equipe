Sobre o Projeto

Site de Equipe

O objetivo deste projeto é criar uma página da equipe usando apenas HTML e CSS. A ideia é praticar a criação de páginas com boa estrutura e aparência, sem usar ferramentas externas. O site mostra quem faz parte do grupo, o que cada um fez e como entrar em contato. Também serve para treinar o trabalho em equipe e a organização das tarefas.

Tecnologias utilizadas:
